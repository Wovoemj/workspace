### 1. 多模型联网参数适配

```python 
payload["enable_search"] = True 

# 千问: 使用 enable_search 参数   
payload["enable_search"] = True 
``` 

### 2. 智能触发机制 
```python 
def should_enable_search(user_message: str, provider: str) -> bool: 
    """智能判断是否开启联网搜索""" 
    user_msg_lower = user_message.lower() 
    
    # 优先检测旅游行程查询 
    if any(keyword in user_msg_lower for keyword in trip_keywords): 
        return True 
    
    # 检测景点查询 
    if any(keyword in user_msg_lower for keyword in scenic_keywords): 
        return True 
    
    # 检测城市名+旅游组合 
    for city in popular_cities: 
        if city in user_msg_lower: 
            if any(travel_word in user_msg_lower for travel_word in ["日游", "天游", "旅游", "旅行"]): 
                return True 
    
    return False 
```
